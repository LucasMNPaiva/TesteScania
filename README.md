#  — Executar Aplicação ClienteApi & TesteScaniaAlgorithms **ASP.NET Core MVC (.NET 8)**

## Requisitos
- **.NET SDK 8.x** instalado  
  Verifique:
  ```bash
  dotnet --version
  ```

## Como executar
```bash
# 1) Restaurar
dotnet restore

# 2) Compilar
dotnet build -c Release

# 3) Rodar
dotnet run 
```

Saída típica (endereços locais):
- HTTP: `http://localhost:5xxx`
- HTTPS: `https://localhost:7xxx`
